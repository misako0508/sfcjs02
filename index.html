練習問題３

  1. Math.random()でランダムに０〜１までの間の数を抽出する
  2. a=Math.random()でaに代入する
  3. a*100で整数部分を１桁から２桁の数字にする
  4. Math.floor(a*100)で小数部分を取り除く
  5. n=Math.floor(a*100)でnを入力すると０以上９９以下の数字がランダムに出てくるようにする
　　

練習問題３　模範解答

  1. let random_value, scaled n を実行
  2. random_value=Math.random();を実行
  3. scaled=random_value*100;を実行
  4. n=Math.floor(scaled);を実行
